        "fuel": 123,
        "gallonsPerThrust": 123,
        "gallonsPerSteer": 123,

#### Stop Simulation
Stops current simulation, allows users to reload it later.

Request:
`GET /simulation/stop`

Response:
```json
200 Success
```

#### List Saved Simulations
Lists the existing simulations from the server.

Request:
`GET /simulations`

Response:
```json
{
    "simulations": [
        {
            "id": 123,
            "name": "[SIMULATION_NAME]"
        },
        {
            "id": 123,
            "name": "[SIMULATION_NAME]"
        }
    ]
}
```
